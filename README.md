# Krushna Thakkar — Portfolio

> Physics-simulation themed portfolio with city-shifting education section, dynamic per-section themes, and GSAP-powered animations.

🌐 **Live:** https://kru2710shna.github.io/WEBISTE/

## Stack
- Vanilla HTML/CSS/JS (zero build step — just open and it works)
- GSAP 3 (ScrollTrigger, animations)
- Canvas API (particle physics, starfield)
- Google Fonts (Bebas Neue, Syne, JetBrains Mono, Playfair Display)

## Sections
| Section | Theme |
|---|---|
| Home | Physics particle simulation, floating ML formulas |
| About | Terminal-style card |
| Education | City backgrounds shift per school (Sacramento → SF → San Jose → Pittsburgh) |
| Experience | Dark editorial, green accents |
| Projects | Brutalist red, 12-col asymmetric grid |
| Contact | Cosmic starfield canvas |

## Run locally
```bash
open index.html
# or
python3 -m http.server 3000
```
